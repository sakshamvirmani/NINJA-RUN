# Ninja-Run

A Python side-scroller built with Pygame. Run, jump, and dodge hurdles — built from scratch with a hand-rolled physics engine, character movement logic, and frame-by-frame sprite animation.

## Features

- Custom physics engine — gravity, jump velocity, and ground collision handled manually
- Sprite animation — 5-frame dragon character cycle for smooth movement
- Procedural hurdle generation — randomized obstacle spawning with increasing difficulty
- Collision detection — pixel-accurate bounding-box checks

## Tech Stack

- Python 3
- Pygame

## Run Locally

```bash
pip install pygame
python "Sam Ninja Run Game Project 1.py"
```

## Controls

| Key | Action |
|-----|--------|
| `Space` / `↑` | Jump |
| `Esc` | Quit |
